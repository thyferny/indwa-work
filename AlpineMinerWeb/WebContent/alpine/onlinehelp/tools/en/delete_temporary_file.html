Delete Temporary File

Through settings in the configuration file sys.properties in ALPINE_DATA_REPOSITORY\preferene\ administrators can set the period for storing temporary files and how often to execute the clean-up task. The values are measured in milliseconds. Following are the configurations in sys.properties:

temp_file_livetime = 86400000
temp_file_scan_frequency = 86400000
temp_file_livetime: how long to store temp files. (Default is 1 day.)
temp_file_scan_frequency: interval between cleaning tasks..

Updated values will take effect when Alpine is restarted.
