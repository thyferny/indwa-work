Installation

 

Environment

Alpine Data Labs suggests using Firefox 7 and above or IE 8 and above to access Alpine Illuminator. Other web browsers are also supported, but have not been tested for full compatibility.

 

Prerequisite

The installer requires Apache Tomcat to be installed. The application will also work with other web application servers; however, it has not been tested on other servers. Manual installation will be required for running the application on other web server.
Please make sure Tomcat can be started and the ${tomcatpath}/bin/startup.sh has the executable permission.

 

Install on Mac or Linux

The installer is a self-extracting shell script. i.e. alpine_illuminator_installer.bin
The installer must be run by the root user.

Installation steps

  1. Accept the license agreement.
  2. Input Tomcat installation path (no default).
  3. Input directory name for storing data.
  4. Restart tomcat?[y/n], default is yes.

Installation in Windows

Unzip the package containing Alpine Illuminator. Run install_illuminator_windows.bat to install on Windows.
The batch command file will ask for the Tomcat install path.

 

Post Installation

After installation the user can access the application using the flowing web address http://localhost:8080/AlpineIlluminator. There are two default user accounts. admin/alpineadmin and guest/guest. After login as admin/alpineadmin please perform the following tasks:

  • Update DB connection settings.
  • Configure email settings.
  • Reset admin password.

 

Browser Setting

The following browser settings will allow results to open in a new tab.

IE

To change setting on IE Browser, following below steps:

  1. Open Tool menu item.
  2. Click Internet Options window.
  3. Click the Settings button within the Tabs section.
  4. Select the “Always open pop-ups in a new tab” option in When a pop-up is encountered
  5. Save the change.

Firefox

To change setting on Firefox Browser, following below steps:

  1. Open a new tab in your Browser and enter “about:config” into the browser’s address bar.
  2. Navigate to browser.tabs.loadDivertedInBackground.
  3. Change its value to true.
  4. Navigate to browser.link.open_newwindow.restriction.
  5. Change its value to 0.
  6. Save the changes. And restart your browser.

Safari

To change setting on Safari Browser, following below steps:

  1. Open Safari menu, and click Preferences menu item.
  2. Open the Tabs section.
  3. Select “Always” for “Open pages in tabs instead of windows.”
  4. Pitch on box of command-click opens a link in a new tab.

Then to open a new tab in the background, click the link (or button) while pressing the command button.
